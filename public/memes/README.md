# Cardano Memes for Subscrybe

## How to Add Memes

1. Save the meme images you have to these folders:

```
/public/memes/
  /loading/
    - morpheus-energy.jpg (Matrix meme about Cardano efficiency)
    - skeleton-waiting.jpg (Waiting for ADA tokens)
  /success/
    - sold-kidneys.jpg (POV: Sold kidneys to buy Cardano)
  /price/
    - investing-under-1.jpg (Investing under $1, seeing price above $5)
  /community/
    - vs-haters.jpg (Me vs Cardano haters - lion vs monkey)
    - blockchain-choice.jpg (Morpheus offering Cardano pills)
```

## Meme Mapping

### 1. morpheus-energy.jpg
**Use for**: Loading states, explaining Cardano benefits
**Caption**: "What if I told you Cardano will be thousands of times more energy efficient than Bitcoin"
**Location**: Welcome modal, about page

### 2. sold-kidneys.jpg
**Use for**: Success state after adding subscription
**Caption**: "POV: Sold my kidneys to buy Cardano"
**Location**: Subscription added success

### 3. investing-under-1.jpg
**Use for**: Budget warnings, price alerts
**Caption**: "Investing in Cardano under 1$ / Seeing price above 5$"
**Location**: Budget warning component

### 4. vs-haters.jpg
**Use for**: Achievement unlocked
**Caption**: "Me vs. Cardano haters"
**Location**: Konami code easter egg

### 5. blockchain-choice.jpg
**Use for**: Welcome screen, platform choice
**Caption**: "POV: You are a blockchain developer looking for a building platform - Cardano / Cardano"
**Location**: Welcome modal

### 6. skeleton-waiting.jpg
**Use for**: Loading states, waiting for transactions
**Caption**: "Developers waiting for ADA tokens"
**Location**: Transaction processing

## Instructions

1. Download/save the meme images from your source
2. Place them in the appropriate folders under `/public/memes/`
3. Name them as indicated above
4. The components will automatically load them
